# Flood Resilience System - Backend

Backend API for the Flood Resilience System built with FastAPI and Python.

## Features

- RESTful API endpoints for flood data
- PostgreSQL database with PostGIS
- Async/await with asyncpg
- OpenAPI documentation
- Authentication and authorization
- Real-time alerts and notifications

## Getting Started

See the main project README for complete setup instructions.

## Usage

```bash
poetry run uvicorn app.main:app --reload
```

The API will be available at `http://127.0.0.1:8000`
